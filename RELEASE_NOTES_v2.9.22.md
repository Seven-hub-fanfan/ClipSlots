# ClipSlots v2.9.22 — UI 全面优化

本次围绕主界面、节点画布、圆盘预览、插件中心与权限弹窗做了一轮系统性的 UI 打磨。

## 主要改动

- **槽位卡片更紧凑**：空槽图标缩小、说明并为一行；有内容卡片按钮区改用小号控件并降低高度；整体卡片高度收紧，让 10 个槽位尽量不滚动就能看全。
- **文本预览更充分**：槽位文本预览行数上限放宽，减少过早省略与空白。
- **节点画布按钮精简**：顶部只保留「串联 · 模板 · 清除 · 完成」四个按钮（串联/清除按范围下拉，模板合并导出与导入，批量应用并入串联菜单），底部操作栏整行删除，视觉更干净。
- **修复圆盘预览不透明色块**：预览面板头部与空态背景改为半透明毛玻璃，消除圆盘弹出后"大块不透明色块遮屏"的问题。
- **「连接」按钮升级**：加入节点连线图标、强调色渐变与描边/投影，更有质感。
- **版本号迁移**：从右下角移动到左上角「检查更新」按钮右侧。
- **辅助权限弹窗优化**：精简文案、加大字号与行间距，减轻密集压迫感。
- **插件中心补「社区 Skill」分类**：与「社区插件」并列（即将开放）。
- **插件图标优化**：改用更精致的拼图图标并加上红点通知。
- **导出连接模板弹窗重做**：头部图标+标题、卡片式范围选项（主次层次分明）、圆角与间距统一，风格更专业。
