# ClipSlots v2.9.21 — 修复节点画布端口消失 + 界面精简与居中

## 修复内容

- **四边端口不再消失**：端口圆点位于卡片上 / 下 / 左 / 右四边外侧，此前鼠标从卡片移向任意一边的端口时会离开卡片 hover 区域，导致端口从就绪态缩回 / 消失。现已将卡片 hover 命中区四向外扩 12px，完整覆盖四个方向的端口圆点，鼠标移向任一边端口都能稳定命中。
- **拖拽连线期间端口恒就绪**：一旦进入拖拽连线模式，所有节点的端口都会保持就绪态，无论 hover 与否，直到连线完成或取消，避免拖拽途中目标端口缩回 / 消失。

## 界面优化

- **去掉顶部副标题小字**：移除标题栏下方「独立画布内编辑连接；主界面继续保持干净，只显示色点提醒。」说明文字，界面更简洁。
- **去掉底部「当前链路」小字**：移除底部操作栏上方的「当前链路：1→6 …」文字行，连线关系从画布本身即可看清，无需文字重复。
- **节点网格居中显示**：节点网格在画布内水平 + 垂直居中，不再紧贴左上边缘。

仅改动节点画布相关文件，不改动槽位数据层与连线数据结构，深浅色模式均正常。
