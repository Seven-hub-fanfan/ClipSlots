# ClipSlots v2.9.21 — 修复节点画布端口消失问题

本次为节点画布连接交互的稳定性修复，专门解决"端口刚要点就找不到"的问题。

## 修复内容

- **四边端口不再消失**：端口圆点位于卡片上 / 下 / 左 / 右四边外侧，此前鼠标从卡片移向任意一边的端口时，会离开卡片 hover 区域，导致端口从就绪态缩回 / 消失。现已将卡片 hover 命中区四向外扩 12px，完整覆盖四个方向的端口圆点，鼠标移向任一边端口都能稳定命中。
- **拖拽连线期间端口恒就绪**：一旦进入拖拽连线模式，所有节点的端口都会保持就绪态，无论 hover 与否，直到连线完成或取消，避免拖拽途中目标端口缩回 / 消失。

仅改动节点画布相关文件，不改动槽位数据层与连线数据结构，深浅色模式均正常。
